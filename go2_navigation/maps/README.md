# Maps Directory

This directory is intended to store the 2D maps generated by SLAM for Nav2.

1. Copy your generated map files (e.g., `map.yaml` and `map.pgm`) into this directory.
2. The default map file expected by the launch file is `map.yaml`. If your map has a different name, you can launch navigation with:

```bash
ros2 launch go2_navigation navigation.launch.py map:=/full/path/to/your_map.yaml
```
